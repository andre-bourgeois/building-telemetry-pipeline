# building-telemetry-pipeline
[![Licence: MIT](https://img.shields.io/badge/Licence-MIT-blue.svg)](LICENSE)

A Brick Schema-annotated sensor network for your living space. It streams to time-series storage via MQTT, with a Grafana dashboard and an anomaly detector running on the edge.
